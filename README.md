# Omi AI Studio

Advanced AI Visual Studio for production-ready images and videos.

## Features

- **AI Video Generation**: Create cinematic shots with start/end frames.
- **AI Image Generation**: State-of-the-art text-to-image synthesis.
- **Character Consistency**: Maintain visual identity across scenes.
- **AI Avatars**: Lifecycle digital humans with natural movement.

## Technologies

- Next.js (App Router)
- React + TypeScript
- Supabase (Auth & Database)
- Google Cloud Run (Deployment)
- Tailwind CSS
- GSAP & Framer Motion

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
